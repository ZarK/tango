================================================
Django Basic Tumblelog
http://code.google.com/p/django-basic-tumblelog/
================================================

A simple tumblelog application for Django projects.

To install this app, simply create a folder somewhere in
your PYTHONPATH named 'basic' and place the 'blog'
app inside. Then add 'tumblelog' to your projects
INSTALLED_APPS list in your settings.py file.

=== Dependancies ===
 * [http://code.google.com/p/django-tagging Django Tagging]
 * [http://pygments.org/ Pygments]
 * [http://www.pythonware.com/products/pil/ Python Imaging Library (PIL)]
